git 常用命令

如果指向克隆最新的版本git clone --depth=1

显示某次提交的内容git show $id

抛弃工作区修改git co --<file>

将工作文件修改到暂存区git add <file>

从版本库中删除文件git rm <file>
从版本库中删除文件，但不删除文件git rm <file> --cached

提交你的修改git commit -m '??'

提交更新到远程服务器git push

从远程服务器，更新合并本地项目

以上命令的含义：
首先从远程的origin的master主分支下载最新的版本到origin/master分支上
然后比较本地的master分支和origin/master分支的差别
最后进行合并
